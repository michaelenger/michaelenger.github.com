Main Loop

  • Loops ... FOREVER
  • Comes in two flavors:
    • Draw
    • Update

Main Loop - Draw

  • Limited FPS
  • Draw your resources
    • Back-to-front
    • Cleared every time
  • DO NOT UPDATE HERE, FFS

Main Loop - Update

  • "Unlimited" speed
  • Use dt (delta time)
  • Update entities/variables/whatevs.
  • DO NOT DRAW HERE, FFS

Input - Callbacks

  • Keyboard
    • key pressed/released
  • Mouse
    • mouse button pressed/released
  • Joystick
    • joystick button pressed/released

Input - Polling

  • Keyboard
    • is key down?
  • Mouse
    • get position
    • is button down?

Input - Polling

  • Joystick
    • axes
    • balls
    • hats
    • is button down/up?

Sprites

Loading an Image

function love.load()
	image = love.graphics.newImage("logo.png")
	x = 100 -- from top left
	y = 100
end

function love.draw()
	love.graphics.draw(image, x, y)
end

Keyboard Movement

function love.keypressed(key)
	if key == "left" then
		x = x - 10
	elseif key == "right" then
		x = x + 10
	elseif key == "up" then
		y = y - 10
	elseif key == "down" then
		y = y + 10
	end
end

Mouse Movement

function love.update(dt)
	x = love.mouse.getX() - (image:getWidth() / 2)
	y = love.mouse.getY() - (image:getHeight() / 2)
end

Smooth Movement

function love.update(dt)
	local target = love.mouse.getX() - (image:getWidth() / 2)
	local diff = target - x
	x = x + (diff * dt)

	target = love.mouse.getY() - (image:getHeight() / 2)
	diff = target - y
	y = y + (diff * dt)
end
